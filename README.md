# 🎮 Video Game Sales Prediction

A Machine Learning project that predicts video game sales using historical market data. The project leverages data preprocessing, exploratory data analysis, feature engineering, and regression models to generate accurate sales forecasts and identify factors influencing game performance.

---

## 📌 Project Overview

This project aims to analyze historical video game sales data and build predictive models that estimate future sales performance. By applying Machine Learning techniques, the system helps uncover market trends and supports data-driven decision-making.

---

## 🚀 Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Regression-Based Sales Prediction
- Model Evaluation and Performance Analysis
- Data Visualization and Insights
